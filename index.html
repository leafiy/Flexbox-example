Flexbox.less easy to use flexbox with LESS

Just a demo for showing how to use Flexbox in the simplest way, without worrying about flex-versins

Support IE10+ / Chrome / Safari / Firefox / Opera(only webkit)

For flex container

Basic css code

display: -webkit-box; display: -webkit-flex; display: -moz-box; display: -ms-flexbox; display: flex;

-webkit-flex-wrap: wrap; -ms-flex-wrap: wrap; flex-wrap: wrap;

If using less makes flex simple

.flex-container{display: -webkit-box; display: -webkit-flex; display: -moz-box; display: -ms-flexbox; display: flex;-webkit-flex-wrap: wrap; -ms-flex-wrap: wrap; flex-wrap: wrap;}

.flex-container;

For each flex items

.flex-item{ display: -webkit-box; display: -webkit-flex; display: -moz-box; display: -ms-flexbox; display: flex; }

.flex-item;

Clear flex attr

.flex-none{flex: none; -webkit-flex: none; -moz-flex: none; -ms-flex: none;}

.flex-none;

Flex item count

.flex-count(@a){ -webkit-box-flex: @a; -webkit-flex: @a; -moz-box-flex: @a; -ms-flex: @a; flex: @a; }

.flex-count("num");

Flex align

.flex-align(@a){ -webkit-flex-align: @a; -ms-flex-align: @a; -webkit-align-items: @a; align-items: @a; -webkit-box-pack: @a; -moz-box-pack: @a; -ms-flex-pack: @a; -webkit-justify-content: @a; justify-content: @a; -webkit-box-align: @a; -moz-box-align: @a; }

@a:center / flex-start / flex-end / space-between / space-around
